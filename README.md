# EXP-3-PROMPT-ENGINEERING-

## Aim: 
Evaluation of 2024 Prompting Tools Across Diverse AI Platforms: 
ChatGPT, Claude, Bard, Cohere Command, and Meta
Experiment:
Within a specific use case (e.g., summarizing text, answering technical questions), compare the performance, user experience, and response quality of prompting tools across these different AI platforms.

## Algorithm:
 1.Select a common use case (e.g., text summarization or answering technical questions).

 2.Prepare the same prompt for all AI platforms.

 3.Enter the prompt into each platform:

ChatGPT

Claude

Bard/Gemini

Cohere Command

Meta AI

4.Observe and record:
  Accuracy of response
  
  Clarity
  
  Response speed
  
  User experience

  Creativity and completeness

5.Compare the outputs from all platforms.

 6.Analyze strengths and weaknesses of each tool.

 7.Document the final result and conclusion.
